Guess-A-Number!

Planet Earth's Favorite Guessing Game

Think you can guess which number I'm thinking of? First try? Doubtful!

How to play:

1. Press OK to start the game (or cancel if you're too chicken!)

2. Now it's time to guess! Guess the number (1 to 100) and type in your answer. Then, press OK.

3. Once you have discovered the correct number, you can play again by saying "yes" or "no" when prompted.

4. HAVE FUN!
